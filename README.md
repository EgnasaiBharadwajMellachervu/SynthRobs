# Circuito Clone

A free, open-source clone of circuito.io for Arduino circuit design, wiring diagrams, and code generation.

## Features

- Drag-and-drop component selection
- Interactive circuit canvas
- Automatic wiring diagrams
- Arduino code generation
- Support for Arduino Uno, Nano, Mega
- 50+ components (LEDs, buttons, sensors, servos, etc.)
- Real-time wiring validation
- Free forever - no signup required

## Tech Stack

- Frontend: HTML5, CSS3, JavaScript (Vanilla)
- Hosting: GitHub Pages
- No external dependencies or frameworks

## Live Demo

Visit: https://[username].github.io/circuito-clone

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. Start designing circuits!

## Project Structure

```
circuito-clone/
├── index.html           # Main HTML file
├── css/
│   └── style.css       # All styling
├── js/
│   ├── main.js         # Main app logic
│   ├── components.js   # Component database
│   ├── wiring.js       # Wiring rules engine
│   ├── canvas.js       # Canvas rendering
│   └── generator.js    # Code generation
└── data/
    └── components.json # Component definitions
```
